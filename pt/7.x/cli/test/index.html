Versão: 7.x

pnpm test

Aliases: run test, t, tst

Executa um comando arbitário especificado na propriedade test do objeto scripts do pacote.

A intenção de usar a propriedade é especificar um comando que executará teste de unidade ou de integração em seu programa.
