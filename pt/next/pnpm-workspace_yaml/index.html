pnpm-workspace.yaml

O arquivo pnpm-workspace.yaml define a raiz do workspace [][] e possibilita que você inclua / exclua diretórios do workspace. Por padrão, todos os pacotes de todos os subdiretórios estão incluídos.

Por exemplo:

pnpm-workspace.yaml
packages:
  # todos os pacotes em subdiretórios diretos de packages/
  - 'packages/*'
  # todos os pacotes em subdiretórios de components/
  - 'components/**'
  # exclui pacotes que estão dentro dos diretórios de teste
  - '!* */teste/**'

O pacote raiz é sempre incluído, mesmo quando os curingas de localização customizado são usados.
